# Nicholas Gagnon — Personal Portfolio

A clean, Apple-inspired personal portfolio website built with vanilla HTML, CSS, and JavaScript.

## Pages

- **Home** — Hero section, stats, about preview, skills, featured project, and project cards
- **About** — Bio, education/experience timeline, skills breakdown, and photo gallery
- **Projects** — Filterable project grid with clickable modal popups for project details

## Features
- Smooth scroll animations on page load and scroll
- Filterable projects by category (ML, Analytics, Strategy)
- Clickable project modals with full detail view (close with ✕ button or Esc key)
- Sticky frosted-glass navigation bar
- Resume download link
- Fully responsive — works on mobile, tablet, and desktop
- Zero external JS dependencies

## Built With

- HTML
- CSS
- JavaScript
